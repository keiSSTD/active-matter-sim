# Active Matter Simulation (ABP Visualizer)

ブラウザで動く「アクティブマター（Active Brownian Particles）」のインタラクティブ・シミュレーションです。
粒子が自発的に集まり、結晶化していく「MIPS (Motion-Induced Phase Separation)」現象を観察できます。

## 🎮 Demo
**実際に遊んでみる (Click here) 👇**
https://keisstd.github.io/active-matter-sim/

## ✨ Features
* **Real-time Interaction:** スライダーで「粒子の速度 (Péclet数)」や「密度」を動的に変更可能。
* **MIPS Visualization:** 密度が高い領域（クラスタ）の粒子は色が変化し、相分離が視覚的にわかります。
* **No Install Required:** HTML5 Canvas + JavaScript で動作するため、インストール不要でスマホでも動作します。
